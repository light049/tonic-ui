# DatePicker

`Datepicker` is used to select a date from a calendar or type in a date value manually. It is composed of `Calendar`, `DateInput`.

## Import

```js
import {
  Calendar,
  DateInput,
  DatePicker,
} from '@tonic-ui/react';
```
## Usage

### Calendar
```jsx
<Calendar />
```

### DateInput
```jsx
<DateInput value="2021.12.15" />
```

### DatePicker
```jsx
<DatePicker value="2021-12-15" />
```
