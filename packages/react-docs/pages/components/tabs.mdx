# Tabs

Tabs is used to group and display a set of related elements.

## Import

```js
import {
  Tab,
  Tabs,
  TabList,
  TabPanel,
} from '@tonic-ui/react';
```

## Usage

```jsx
<Tabs>
  <TabList>
    <Tab>One</Tab>
    <Tab>Two</Tab>
    <Tab>Three</Tab>
  </TabList>
  <TabPanel>
    <p>One</p>
  </TabPanel>
  <TabPanel>
    <p>Two</p>
  </TabPanel>
  <TabPanel>
    <p>Three</p>
  </TabPanel>
</Tabs>
```
