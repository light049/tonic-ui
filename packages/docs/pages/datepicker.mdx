# Date Picker

`Datepicker` is used to select a date from a calendar or type in a date value manually. It is composed of `Calendar`, `DateInput`, and `TimeInput`.

## Import

```js
import {
  DatePicker,
  Calendar,
  DateInput,
} from '@trendmicro/react-styled-ui';
```
## Usage

```jsx
<DatePicker />
```
